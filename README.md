# Face Mesh Detection using MediaPipe

A real-time Face Mesh Detection application built with **Python**, **OpenCV**, and **MediaPipe**. The application captures live video from a webcam and detects facial landmarks, drawing a detailed 3D face mesh in real time.

## 📌 Features

* 🎥 Real-time webcam face detection
* 🧠 MediaPipe Face Mesh with 468 facial landmarks
* ⚡ Fast and lightweight processing
* 🖥️ Cross-platform (Windows, Linux, macOS)
* 🐍 Built using Python and OpenCV

---

## 🛠️ Technologies Used

* Python 3.x
* OpenCV
* MediaPipe

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Narasimha2308/Face-mesh.git
cd Face-mesh
```

### 2. Create a virtual environment

**Windows (PowerShell)**

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

**Linux/macOS**

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python face.py
```

---

## 📸 Output

* Opens the webcam.
* Detects the face in real time.
* Draws a facial mesh with MediaPipe landmarks.
* Press **ESC** to close the application.
